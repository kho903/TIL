# git repo clone으로 시작하기
## github에서 create a new repository
1.  repository name 정하기
2.  Initialize this repository with:
	- Add a README file 체크
	- Choose a license - MIT License
## iterm2 실행
1. 원하는 디렉토리로 이동 
- `cd 원하는 디렉토리`
2. git clone
- `git clone git https://github.com/{아이디}/{레포지토리 이름}`
3. 클론 완료 후 해당 폴더로 이동
- `cd 레포지토리 이름`
